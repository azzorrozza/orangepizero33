# Auditoria - OrangePi Zero 3
## Validação Completa - Docker CE + Portainer

Este documento realiza uma auditoria completa da instalação do Docker CE e do Portainer.

> **Observação:** Todos os comandos abaixo são apenas de leitura. Nenhuma configuração será alterada.

---

# Sistema

```bash
docker --version

docker version

docker compose version
```

---

# Pacotes instalados

```bash
apt policy docker-ce

dpkg -l | grep -E 'docker|containerd|buildx'
```

---

# Serviço Docker

```bash
systemctl status docker --no-pager

systemctl is-enabled docker

systemctl is-active docker

journalctl -u docker -n 100 --no-pager
```

---

# Informações do Docker

```bash
docker info

docker system df
```

---

# Estrutura de diretórios

```bash
tree /opt/docker
```

---

# Rede Docker

```bash
docker network ls

docker network inspect services
```

---

# Volumes Docker

```bash
docker volume ls
```

---

# Containers

```bash
docker ps

docker ps -a

docker compose ls
```

---

# Portainer

```bash
docker compose -f /opt/docker/portainer/docker-compose.yml ps

docker logs --tail 100 portainer

docker inspect portainer
```

---

# Arquivo docker-compose

```bash
cat /opt/docker/portainer/docker-compose.yml
```

---

# Teste do Docker Hub

```bash
docker run --rm hello-world
```

---

# Recursos utilizados

```bash
docker stats --no-stream
```

---

# Armazenamento

```bash
docker system df -v
```

---

# Configuração do Docker

```bash
cat /etc/docker/daemon.json 2>/dev/null || echo "daemon.json não configurado"
```

---

# Diretório de dados

```bash
docker info | grep "Docker Root Dir"
```

---

# Eventos recentes

```bash
journalctl -u docker --since "1 hour ago" --no-pager
```

---

# Portas em uso

```bash
ss -lnpt | grep -E '9443|8000|2375|2376'
```

---

# Firewall

```bash
iptables -L -n -v

ip6tables -L -n -v
```

---

# Estado geral do sistema

```bash
systemctl --failed
```

---

# Uso de recursos

```bash
free -h

df -h

uptime
```

---

# Resultado esperado

Ao final da auditoria, espera-se encontrar:

- ✅ Docker CE instalado a partir do repositório oficial.
- ✅ Docker Engine em execução e habilitado no boot.
- ✅ Docker Compose Plugin instalado.
- ✅ Buildx instalado.
- ✅ Containerd em execução.
- ✅ Network `services` criada.
- ✅ Container Portainer em execução.
- ✅ Portainer conectado à rede `services`.
- ✅ Estrutura `/opt/docker` organizada.
- ✅ Teste `hello-world` executando com sucesso.
- ✅ Nenhum serviço com falha (`systemctl --failed`).
- ✅ Firewall contendo as regras do Docker e do Tailscale.
- ✅ Porta 9443 em escuta.
- ✅ Recursos do sistema dentro do esperado.
